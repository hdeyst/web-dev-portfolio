# Portfolio Webpage 

## Projects
This website highlights projects that I have made at UVM in some of my CS classes. The project I completed for Software Engineering, as well as the one I did for Mobile App Development are both displayed on the "CS Projects" page. 
- In order to showcase the mobile app on the website, I used some screenshots from the active application, and created a slideshow presentation of them, which can be toggled through by the user.
- Each project shown links to the corresponding Github page.


## Animation
In order to create the animation that is the background of each page, I used JavaScript to manipulate my own images and move them across the screen. The code for this is in the file `fishAnimation.js`.

## Contact Page
The contact page utilizes HTML's form tag. The form checks to make sure a user is including their email and name before they are able to submit it.

## About Me
The about me page pulls from a JSON file that I wrote. Storing data in a separate JSON file helps when writing clean and maintainable code. A corresponding javascript function fetches the JSON data and displays it nicely onto the homepage. 

## Navigation
There is a simple navigation system that links the user to each page of the website. In addition, the icon in the corner will take the user to the homepage, CS Projects, from any location on the page.
